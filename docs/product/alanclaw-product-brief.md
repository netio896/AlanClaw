# AlanClaw 产品定位简报

## 产品定义

AlanClaw 是一个面向缅甸市场的 AI Agent 产品，底层可对接 OpenClaw/Hermes 一类执行与记忆能力，上层提供适合本地商业场景的专家广场、客服自动化、内容生产和办公协同体验。

## 当前阶段目标

当前阶段先完成专家广场内容底座，让产品具备：

- 可展示的专家目录
- 可导入后台的结构化数据
- 可维护的专家详情源文件
- 可演进到前台/后台应用的仓库结构

## 核心受众

- 缅甸本地中小企业
- 跨境销售与客服团队
- 内容运营团队
- 需要多语言协同的业务团队

## v1 专家能力方向

- 销售与客服
- 内容营销
- 办公协同
- 文档处理
- 数据处理
- 市场研究
- 广告投放
- 知识管理

## 支持语言

- Burmese
- English
- Chinese

## 预期支持渠道

- Telegram
- Facebook / Messenger
- Email
- Web
- File upload
- TikTok（内容侧）

## 产品原则

- 中文优先整理内容资产，便于内部策划与运营
- 输出直接可用于广场卡片、详情页和系统提示词
- 结构化优先，避免后续重复整理
- 内容与产品实现分层，先稳住内容源，再接入应用
