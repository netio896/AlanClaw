# AlanClaw v0.2: Myanmar Agent Team + BIM Tools + HF Space Demo

AlanClaw v0.2 把项目从本地专家广场，推进到可展示、可安装、可演示的 Myanmar Agent Team 产品包。本版本重点不是新增复杂后端，而是把“行业专家团队”这条主线整理成可以给用户、合作方和团队成员看的交付物。

## Release Highlights

- **Myanmar Agent Team Demo**
  - 新增 Hugging Face Space Demo 源码：`spaces/alanclaw-myanmar-agent-team`
  - 提供公开演示入口，用于展示 AlanClaw 的团队式 Agent 概念
  - Demo 不依赖 API Key，适合做产品预览、销售演示和早期用户沟通

- **Industry Agent Team Direction**
  - 继续强化 AlanClaw 的核心定位：从单个专家卡片升级为行业 Agent Team
  - 当前团队方向覆盖海外业务、建筑行业、电商增长、缅甸社媒营销、酒店推销获客等场景
  - Web/Admin 侧保留现有数据结构，避免为了演示破坏核心维护链路

- **BIM / Construction Tooling Skill**
  - 新增 `skills/alanclaw-bim-tools`
  - 将两个 Hugging Face Spaces 纳入 AlanClaw 建筑行业团队工作流：
    - IFC Hierarchy Viewer: `https://huggingface.co/spaces/network01/ifc-hierarchy-viewer`
    - Floor Plan Detection: `https://huggingface.co/spaces/network01/Floor-Plan-Detection`
  - 明确 IFC 模型、平面图图片、BOQ 前置检查、人工复核提示等使用边界

- **Public Demo Readiness**
  - 新增 `spaces/README.md`，说明 Space 目录用途和部署方式
  - 新增 Space README、Gradio app、图标资源和依赖文件
  - 方便后续把 AlanClaw 的行业团队能力发布到 Hugging Face 或其他演示平台

## What Users Can Try

1. 打开 AlanClaw Web 专家广场，浏览行业团队和专家卡片。
2. 选择一个行业团队，例如建筑行业团队或酒店推销获客团队。
3. 一键添加推荐团队，进入“我的专家”查看团队成员。
4. 打开 Hugging Face Space Demo，体验 AlanClaw Myanmar Agent Team 的公开演示流程。
5. 对建筑/BIM 场景，使用 IFC Viewer 或 Floor Plan Detection 的结果，再交给 AlanClaw 建筑行业团队整理成可沟通的业务输出。

## Notes

- 本版本仍是本地开发产品包，不包含登录、权限、数据库或正式云端部署。
- Hugging Face Demo 用于产品展示，不等同于完整生产后端。
- BIM/Floor Plan 工具输出只能作为前置分析和沟通材料，不能作为最终工程数量、报价、合同或安全结论。
